Vim学习笔记
====================


## 技巧


###字符操作
###基础
* i  进入Insert模式
* o  进入Insert模式并换行
* r  replace 替换
* a  append 追加
* p  黏贴于光标后
* P  黏贴于光标前
* visual 模式下面的字符串选择
####进阶
* ]p 自動indent的黏貼
* :n1,n2 m n3     移动n1-n2行(包括n1,n2)到n3行之下；
* :n1,n2 co n3    复制n1-n2行(包括n1,n2)到n3行之下；
* :n1,n2 d        删除n1-n2行(包括n1,n2)行；
